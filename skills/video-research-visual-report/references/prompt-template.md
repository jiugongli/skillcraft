# Reusable Prompt

Please perform deep video understanding, multi-source research, adversarial analysis, and visual reporting for this video:

Video link:
[paste link]

Optional parameters:

- Target audience: [general / expert / investor / developer / student / custom]
- Output language: [Chinese / English / bilingual]
- Output format: [auto / long image / multiple short images / PPT / text review / all]
- Visual style: [information report / magazine / educational / business analysis / minimal]
- Video screenshots: [allowed / not allowed / reference only]
- Rigor level: [normal / high / academic-ish]
- Use case: [social post / article / deck / internal research / custom]

Requirements:

1. Get subtitles or transcript. If unavailable, transcribe audio.
2. Find and read original sources cited by the video.
3. Browse official sources, third-party analysis, community feedback, and dissenting views.
4. Do not fully trust the video author or any single source.
5. Separate facts, claims, speculation, and uncertainty.
6. Organize supporting, opposing, and conditional views.
7. Perform 2-3 adversarial passes before concluding.
8. Give practical conclusions by scenario.
9. Decide the visual output format based on information density.
10. Produce clear information images/pages with accurate text.
11. If making a long image, include a `00` explainer page and check for cropped text.
12. Deliver text review, output paths, source links, and uncertainty.
