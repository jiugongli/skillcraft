# Output Format Guide

Choose output format by content shape.

## Short image set

Use when the video has a simple argument or one main takeaway.

Typical size: 3-6 images.

## Long image

Use when the user wants social sharing or a single scrollable report.

Always include:

- `00` explainer page
- safe side margins
- preview image
- edge/crop check

## PPT-style pages

Use when the content is for presentation, internal review, or step-by-step speaking.

Rules:

- One core point per page
- Avoid dense paragraphs
- Use comparison, conclusion, and recommendation pages
- Page count should follow content, not a fixed target

## Multiple chapters

Use when one long image would become too tall or visually crowded.

Example:

- Part 1: claims and sources
- Part 2: evidence and disputes
- Part 3: conclusion and recommendations
